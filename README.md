# Документация инфраструктуры: ServeHub-2
[Гайд по настройке](https://github.com/canntstand/ServeHub-2/blob/main/SETUP_GUIDE.md)

## Цели проекта
1. **Полная приватность**: Хранение всех личных данных и переписки на собственном железе.
2. **Доступность 24/7**: Безопасный доступ к медиатеке, файлам и инструментам управления из любой точки мира.
3. **Единая экосистема**: Использование стандартизированных мобильных клиентов, интегрированных с личными сервисами.
4. **Оптимизация**: Использование домашнего локального сервера в связке с легким VPS для прозрачного туннелирования, обеспечивающего доступ без "белого" IP дома.

## Основные сервисы
- **Nextcloud**: Облачное хранилище для файлов, контактов и фото.
- **Vaultwarden**: Легковесный менеджер паролей (совместим с Bitwarden API) для хранения шифрованных данных.
- **Matrix (Synapse)**: Защищенный сервер обмена сообщениями.
- **Navidrome**: Стриминг личной музыкальной коллекции.
- **Audiobookshelf**: Сервер для аудиокниг и подкастов с синхронизацией прогресса.

## Сетевая архитектура и безопасность
- **Webnames**: Регистратор доменных имен. Используется для владения доменом и управления DNS-записями (A-записи, Wildcard), необходимыми для работы SSL.
- **WG-easy + AmneziaWG**: Шифрованный туннель между устройствами и VPS, скрывающий инфраструктуру. Весь доступ к сервисам возможен строго из подсети VPN `10.8.0.0/24` + веб-панель для управления.
- **Nginx (Proxy)**: Выполняет роль обратного прокси, блокируя любые внешние запросы, не принадлежащие внутренней сети туннеля.
- **SSH**: Настройка доступа по ключам `ed25519` с отключением входа по паролю. Проброс порта `2222` через VPS позволяет безопасно управлять локальным сервером из любой сети.
- **Certbot**: Автоматизация получения SSL-сертификатов (Wildcard) через проверку DNS-01, интегрированную с API Webnames.
- **Fail2ban**: Защита от брутфорс-атак на системном уровне.

## Мониторинг и контроль
- **Prometheus & Exporters**: Сбор метрик хоста (Node Exporter), контейнеров (cAdvisor) и самого веб-сервера (Nginx Exporter).
- **Grafana**: Визуализация состояния системы и производительности сервисов.
- **Uptime Kuma**: Мониторинг аптайма эндпоинтов с интеграцией через `heartbeat_to_vps`, который ежеминутно отправляет push-статусы о работе домашнего сервера.
- **Portainer**: Веб-интерфейс для администрирования Docker-контейнеров.

## Алертинг и коммуникация
- **Alertmanager**: Фильтрация и маршрутизация критических событий.
- **Matrix Integration**: Использование специального бота для отправки уведомлений от Alertmanager в закрытую комнату личного мессенджера.

## Резервное копирование
- **BorgBackup + Borgmatic**: Шифрованное дедуплицированное резервное копирование.
- **Стратегия**: Автоматическое создание дампов баз данных (PostgreSQL) перед выполнением бэкапа с соблюдением политики хранения версий.
