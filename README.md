## Цели проекта
1. Полная приватность: Хранение личных файлов и переписки на собственном железе.
2. Доступность 24/7: Доступ к медиатеке и документам из любой точки мира.
3. Единая экосистема: Удобные мобильные клиенты для всех сервисов.
4. Минимальные затраты на инфраструктуру: Использование существующего домашнего ПК и прозрачного туннелирования через недорогой внешний VPS.

## Хранение и синхронизация
- Nextcloud: Основное облако для файлов, контактов и фото. Замена Google Drive/iCloud.
- Vaultwarden: Легковесный self-hosted менеджер паролей (реализация Bitwarden API). Хранение паролей, заметок и банковских карт в зашифрованном виде. Совместим со всеми официальными клиентами Bitwarden.

## Коммуникации
- Matrix (Synapse): Защищенный сервер обмена сообщениями.
- Element: Кроссплатформенный клиент для Matrix (семейный мессенджер).

## Медиа-сервисы
- Navidrome: Стриминг личной музыкальной коллекции.
- Audiobookshelf: Сервер для аудиокниг и подкастов с сохранением прогресса прослушивания.

## Сеть и доступ
- FRP (Fast Reverse Proxy): Безопасный проброс локальных веб-сервисов (HTTP/HTTPS) и портов (SSH) через внешний VPS с поддержкой PROXY-протокола для передачи реальных IP-адресов клиентов.
- Docker: Контейнеризация всех сервисов для быстрой развертки и изоляции.

## Мониторинг и контроль
- Prometheus: Сбор метрик производительности системы и контейнеров.
- Grafana: Визуализация состояния сервера (нагрузка на CPU, RAM, занятое место).

## Резервное копирование
- BorgBackup + Borgmatic: Шифрованное дедуплицированное резервное копирование с гибким расписанием. Автоматическое создание дампов баз данных перед бэкапом. Реализация стратегии 3-2-1 (локальная копия + удалённая копия на внешнем носителе или удалённой машине).
