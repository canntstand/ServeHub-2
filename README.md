# Документация инфраструктуры: ServeHub-2

- [Гайд по настройке](https://github.com/canntstand/ServeHub-2/blob/main/SETUP_GUIDE.md)

## Цель проекта
Основная цель — развернуть серверную интернет-архитектуру, которая решает мои личные задачи без значительных финансовых вложений. В её основе лежат следующие сервисы:

- **Nextcloud** — локальное семейное облачное хранилище;
- **Vaultwarden** — полностью безопасное хранилище паролей;
- **Synapse** — защищённый сервер обмена сообщениями, замена публичным мессенджерам для семьи;
- **Navidrome** — стриминг любимой музыки, не занимающей место на устройствах и гарантированно сохранной;
- **Audiobookshelf** — аналогичное решение для книг и аудиокниг.

В процессе проект приобрёл несколько дополнительных задач:

- хранение всех личных данных и переписок исключительно на собственном оборудовании;
- полностью безопасный доступ к медиатеке, файлам и инструментам управления из любой точки мира (с помощью VPN-сети и SSH);
- полный контроль и удобство эксплуатации, обеспечиваемые системой мониторинга, алертинга, резервного копирования и вспомогательных сервисов;
- быстрое автоматизированное развёртывание всей инфраструктуры.

Кроме того, проект одинаково удобен как при использовании с ПК (через веб-интерфейс), так и с мобильных устройств через клиентские приложения.

## Почему решил сделать
1. Мне не хватало определённых сервисов в повседневной жизни, а существующие аналоги по разным причинам меня не устраивали. Я хотел, чтобы нужные мне инструменты работали онлайн, но при этом оставались под моим полным контролем.
2. Разместить всё на одном мощном VPS было бы проще, однако это потребовало бы от меня слишком больших расходов на оплату такого сервера. Поэтому я выбрал гибридный подход: ресурсоёмкие сервисы работают на локальной машине, а удалённый VPS помогает организовать безопасный доступ к ним из интернета.

## Основные сервисы
- **Nextcloud:** Облачное хранилище для файлов, контактов и фото.
- **Vaultwarden:** Легковесный менеджер паролей (совместим с Bitwarden API) для хранения шифрованных данных.
- **Matrix (Synapse):** Защищённый сервер обмена сообщениями.
- **Navidrome:** Стриминг личной музыкальной коллекции.
- **Audiobookshelf:** Сервер для аудиокниг и подкастов с синхронизацией прогресса.

## Сетевая архитектура и безопасность
- **Webnames:** Регистратор доменных имён. Используется для владения доменом и управления DNS-записями (A-записи, Wildcard), необходимыми для работы SSL.
- **WG-easy + AmneziaWG:** Зашифрованный туннель между устройствами и VPS, скрывающий инфраструктуру. Доступ к сервисам возможен строго из подсети VPN `10.8.0.0/24`; также имеется веб-панель для управления.
- **Nginx (Proxy):** Выполняет роль обратного прокси, блокируя любые внешние запросы, не принадлежащие внутренней сети туннеля.
- **SSH:** Настройка доступа по ключам `ed25519` с отключением входа по паролю.
- **Certbot:** Автоматизация получения SSL-сертификатов (Wildcard) через проверку DNS-01, интегрированную с API Webnames.
- **Fail2ban:** Защита от брутфорс-атак на системном уровне.

## Мониторинг и контроль
- **Prometheus & Exporters:** Сбор метрик хоста (Node Exporter), контейнеров (cAdvisor) и самого веб-сервера (Nginx Exporter).
- **Grafana:** Визуализация состояния системы и производительности сервисов.
- **Gatus:** Мониторинг аптайма локального сервера через ежеминутное опрашивание `healthcheck` эндпоинта.
- **Portainer:** Веб-интерфейс для администрирования Docker-контейнеров.

## Алертинг и коммуникация
- **Alertmanager:** Алертинг критических событий на локальном сервере.
- **Gatus:** Мониторинг и алертинг работы локального и удаленного серверов, а также проверка SSL сертификатов.
- **SMTP Integration:** Использование SMTP протокола для отправки уведомлений от Alertmanager и Gatus по почте.

## Резервное копирование
- **BorgBackup + Borgmatic:** Зашифрованное автоматизированное резервное копирование на диск с сжатием lz4.
- **BorgUI:** Интерфейс для удобного просмотра бэкапов.

## Автоматизация
**Ansible:** Система управления конфигурациями для автоматического развёртывания ОС, настройки безопасности и оркестрации Docker-контейнеров.
**Vagrant:** Быстрое развертывание виртуальных машин для тестирования Ansible плейбуков.
