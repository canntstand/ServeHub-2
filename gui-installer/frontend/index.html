Пункт 1. Полный деплой

                                Кнопка необходима, когда деплой временно приостанавливается модулем ansible.builtin.pause для выполнения ручных шагов.
                                


                                Например, при настройке WireGuard и подготовке папок, скрипт выведет инструкцию в окно логов выше и будет ожидать сигнала от этой кнопки для продолжения автоматической работы.
