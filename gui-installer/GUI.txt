done 1. пользователь качает .exe, запускает
done 2. проверка есть ли docker и docker compose, если нет - ошибка (+ кнопка проверить заново), если есть - то кнопка продолжить
done 3. Копирование репозитория рядом со скриптом через git
done 4. Форма заполнения secrets.yml, (просто переменная - рядом описание - и место ввода)
done 5. После нажатия продолжить в пред. шаге все что ввел пользователь должно показываться в окошке с просьбой скопировать и проверить (кнопка продолжить + еще раз спросить точно ли все готово + кнопка переделать) 
done 6. Полученные данные должны парситься в ./репозиторий/ansible/vars/secrets.yml превращаясь в yaml файл
done 7. потом окно выбора разворачивания контейнера ansible через docker (логика такая же как в manage_deploy.sh)
done 8. После подтверждение выбора и запуск деплоя (логи должны выводиться экране)
9. Когда все завершилось писать что все готово, если ошибка то писать что ошибка (+ логи всегда на экране)
